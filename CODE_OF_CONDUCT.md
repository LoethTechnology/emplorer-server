# Code of Conduct

## Purpose

Emplorer is an open-source project intended to support clear, accountable discussion around company reviews. The community around this repository should reflect that same standard.

This code of conduct defines the behavior expected from contributors, maintainers, and anyone participating in project spaces.

## Expected behavior

Participants are expected to:

- communicate respectfully and professionally
- assume good intent while still giving clear technical feedback
- welcome constructive disagreement without turning it into personal conflict
- keep discussions focused on improving the project
- respect differences in background, identity, experience, and perspective
- accept correction when their behavior creates friction or harm

## Unacceptable behavior

The following behavior is not acceptable in project spaces, including issues, pull requests, discussions, and other communication channels:

- harassment, intimidation, threats, or personal attacks
- hateful, discriminatory, or demeaning language
- sexualized language or unwanted sexual attention
- trolling, baiting, or repeated antagonistic behavior
- publishing private or identifying information about another person without consent
- sustained disruption of technical discussion or contribution workflows
- misrepresenting another person's work, intent, or identity

## Scope

This code of conduct applies to all project-managed spaces and to public interactions where a person is representing the Emplorer project or its maintainers.

## Reporting

If you experience or witness behavior that violates this code of conduct, report it privately to the maintainers through the repository's available contact method or issue management process.

When reporting, include:

- what happened
- where it happened
- when it happened
- any supporting context that can help review the incident

## Enforcement

Project maintainers are responsible for interpreting and enforcing this code of conduct. They may take any action they consider appropriate, including:

- requesting a change in behavior
- removing comments, posts, or other contributions
- temporarily limiting participation
- banning a participant from project spaces

Maintainers will aim to review reports fairly, consistently, and with respect for the people involved.

## Project standard

Emplorer exists to encourage more transparent and thoughtful review culture. The standard for this community is simple: strong technical debate is welcome, disrespect is not.
